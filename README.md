# SocialParser

A small library for parsing out common social elements such as hashtags, mentions and urls.

# Todo

## Installation

Add `social_parser` to your list of dependencies in `mix.exs`:

  ```elixir
  def deps do
    [{:social_parser, "~> 0.1.0"}]
  end
  ```
